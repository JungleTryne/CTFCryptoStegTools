# Usage
```bash 

python3 crypto.py [файл с текстом/зашфрованным сообщением].txt [-option1] key.txt

Options:
1)Шифрование цезарем: -Ce key
2)Дешифрование цезарем: -Cd key

4)Шифрование Виженером: -Ve key

7)Дешифрование Вернама: -VVd key
